The Wizard Tutorial is an interactive walkthrough built into Tweenly. It guides you through the core interface — Gallery, Editor, objects, and saving — step by step with highlighted UI elements.

## Starting the tutorial

1. Open the **main menu** (hamburger icon in the top bar).
2. Select **Tutorial**.
3. The tutorial starts in the Gallery and walks you through the key sections.
4. Click **Next** to continue through each step, or **Dismiss** to exit.

:::tip
You can restart the tutorial anytime from the main menu.
:::

## What the tutorial covers

The Wizard Tutorial introduces:

1. **Gallery** — how to find and manage your graphics files.
2. **Creating a new graphic** — opening the Editor with a blank canvas.
3. **Adding objects** — placing rectangles, text, and images on the canvas.
4. **Editing properties** — using the Right Sidebar to adjust styles and positioning.
5. **Saving your work** — the manual save process and metadata fields.

The tutorial highlights each UI element as you go, so you can see exactly where things are.

## What the tutorial does NOT cover

The Wizard covers the basics only. For deeper topics, continue with:

- [Create Your First Graphic](/getting-started/create-your-first-graphics/) — hands-on step-by-step (including animations and export)
- [Animations & Timeline](/user-guide/animations-timeline/) — animation types, timing, easing, labels
- [Data Source & API Control](/main-features/data-source-api-control/) — connecting live data
- [From Design to Broadcast](/getting-started/design-to-broadcast/) — the complete workflow from design to playout

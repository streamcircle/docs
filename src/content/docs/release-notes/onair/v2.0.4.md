## Persistent Runtime

- **Graphics survive page refresh** — when you refresh the browser, On Air reconnects and restores the current graphic state automatically. No more lost graphics on accidental refresh.
- **Tab and session independence** — closing a browser tab or logging out no longer stops running graphics. The runtime continues independently of the UI session.
- **Multi-tab synchronization** — opening On Air in multiple tabs shows the same live state. Actions in one tab are immediately reflected in all others.
- **Automatic reconnect** — after a network interruption, the UI automatically reconnects and synchronizes to the current runtime state.

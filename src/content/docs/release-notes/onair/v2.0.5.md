title: "On Air v2.0.5"
description: Per-data-source refresh, data source caching, URL error handling, and session fixes.
---

*Release date: 04.06.2026*

## New features

- **Per-data-source refresh frequency** — each data source can now have its own refresh interval, allowing independent update rates for different data feeds during live playout.
- **Data source caching** — data source responses are now cached locally, reducing redundant network requests and improving playout performance with frequently accessed data.
- **Logout confirmation** — logging out now requires confirmation in a dialog, preventing accidental session termination during live operation.
- **URL error handling** — On Air now displays clear error messages when an invalid or unreachable URL is entered, instead of failing silently.

## Bug fixes

- **Session management:** Fixed an issue where closing a session was not properly registered, which could lead to stale session state on reconnect.

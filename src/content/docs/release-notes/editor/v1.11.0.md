*Release date: 30.06.2025*

## Timeline redesign

- **New playhead**: The playhead now sits directly above timestamps and animation blocks, showing the current position more clearly.
- **Redesigned timeline controls**: Updated UI and interaction logic for key actions:
    - Timecode input is now fully supported.
    - Simplified playback controls (play, pause, skip).
    - Refined settings for onInit, onEnd, and repeats.
    - Improved display and visibility options.

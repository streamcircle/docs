*Release date: 21.05.2026*

## Pencil Import

- **Import graphics from Pencil** — you can now import `.pen` files directly into the Tweenly Editor. Layers, texts, images, and shapes are preserved from the original Pencil design.
- **Auto-animate** — imported graphics automatically receive animation presets, so you can preview and fine-tune them right away without manually assigning animations.

## Bug fixes

- **Pencil import:** Fixed rendering issues with specific graphic compositions during import.
- **HTML export:** Fixed single quote (`'`) handling in HTML meta export — apostrophes in text content are now exported correctly.

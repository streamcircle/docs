*Release date: 26.03.2026*

## New features

- **Expand / Collapse All in Objects Tree** — expand or collapse all objects in the tree at once.
- **Animations sorted by start time** — animations are now ordered by start time in the list.

## Improvements

- **New fonts: Manrope & Inter** — the editor interface now uses Manrope and Inter fonts.
- **Performance** — optimized internal updates when working with object properties.

## Bug fixes

- **Playback:** The **Next** button at position 0 now correctly starts playback instead of jumping to the next marker.
- **Timeline:** Fixed inverted expand/collapse logic for timeline rows.
- **Fonts:** Manrope font now loads correctly in both the web and graphics-editor environments.
- **Canvas:** Disabled double-click on the canvas, which was causing unintended actions.
